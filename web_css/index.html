基礎知識

CSS(Cascading Style Sheets)とは、Webページの装飾やレイアウトを記述するための言語です。HTML文書の構造とスタイルを分離させる目的で提唱されました。CSSを利用することによって、HTMLで作成された文書の構造を変えることなく、Webページのスタイルを変えることができます。

Web関連技術の標準化を目的とした非営利団体であるW3C(World Wide Web Consortium)が、HTMLやXML(Extensible Markup Language: 文書やデータの構造を記述するための言語)などの他、CSSの仕様も策定しています。
また、HTML5の策定においては、Web関連技術の開発に興味を持つ人々のコミュニティであるWHATWG(Web Hypertext Application Technology Working Group)がW3Cと協力しています。

link要素でHTMLに組み込む

link要素にmedia属性を指定すると、読み込んだ外部スタイルシートを適用するメディアを限定することができます。適用対象の機器の種類だけを限定したい場合には次のキーワードが指定できます。（カンマ区切りで複数指定できます。）

media属性に指定できるキーワード
キーワード 適用対象となるメディア
all すべての機器（デフォルト値）
screen PCやスマートデバイスなどの画面
print プリンタ
projection プロジェクタ
tv テレビ
handheld 携帯用機器
tty 文字幅が固定の端末（テレタイプやターミナルなど）
speech スピーチ・シンセイサイザー（音声読み上げソフト）
braille 点字ディスプレイ
embossed 点字プリンタ

@importでCSSに組み込む

@importの書式を指定することで、CSSの中でさらに外部スタイルシートを読み込ませることもできます。

もっとも簡単な方法は、次のように@importのあとにURLを文字列として指定する方法です。ダブルクォーテーションまたはシングルクォーテーションが必要です。


@import "style.css";

次のような関数形式で指定する方法もあります。書式は異なりますが、機能的にはまったく同じになります。なお、関数形式の場合はダブルクォーテーションやシングルクォーテーションは省略可能です。


@import url(style.css);

URLのあとに、link要素やstyle要素で指定可能なメディア型やメディアクエリを指定することもできます。


@import "style.css" screen, print;

CSSで長さを表す単位には相対的な長さを表す単位と絶対的な長さを表す単位の2種類あります。パーセンテージ（%）については、仕様書では長さの単位とは別扱いになっているのでここには記載しません。

フォントに対する相対的な長さを表す単位
単位 説明
em 要素のフォントサイズを1とする単位
rem html要素のフォントサイズを1とする単位
ex 要素のフォントのx-height（小文字xとほぼ同じ大きさ）を1とする単位
ch 要素のフォントの0の幅を1とする単位（chはcharacterの略）
ビューポートに対するパーセンテージで表す単位
単位 説明
vw ビューポートの幅の1%を1とする単位
vh ビューポートの高さの1%を1とする単位
vmin ビューポートの幅と高さのうち、短いほうの1%を1とする単位
vmax ビューポートの幅と高さのうち、長いほうの1%とする単位
絶対的な長さを表す単位
単位 説明
px ピクセル（1/96インチを1とする単位）
pt ポイント（1/72インチを1とする単位）
pc パイカ（12ポイントを1とする単位）
in インチ（2.54センチメートルを1とする単位）
cm センチメートル
mm ミリメートル
q quarter-millimeters（1/4ミリメートルを1とする単位）

補足説明

CSS3では、プロパティの値として長さや数値が指定できる場合には、それらの代わりにcalc()関数による計算式を指定することもできます。()内に書く計算式には単位の付けられた数値と単位のない数値の両方が使用でき、演算子としては+-*/が使えます。（+と-の前後には必ず空白文字を入れる必要があります。）


font-size: calc(100vw / 35);
margin: calc(1rem - 2px) calc(1rem - 1px);

セレクタ

セレクタを構成する基本的な最小単位のことをシンプルセレクタといいます。たとえば、要素名だけで指定するセレクタもシンプルセレクタの1つです。シンプルセレクタは全部で6種類あり、それらは続けて記入して組み合わせることができます。

ただし、先頭には必ずタイプセレクタ（要素名）またはユニバーサルセレクタ（*）のどちらかを配置するというルールがあります。それ以降にはそれら以外のシンプルセレクタを順不同で必要なだけ記述できます。その際、ユニバーサルセレクタのあとに他のシンプルセレクタが続く場合に限り、ユニバーサルセレクタを省略することが可能です。
たとえば、「.hoge」というセレクタは「*.hoge」のユニバーサルセレクタを省略した書き方です。

組み合わせたシンプルセレクタは、結合子と呼ばれる記号（> + ~）または空白文字で区切って、さらに必要なだけ組み合わせることが可能です。

タイプセレクタ

要素名をそのまま使って適用先を示すシンプルセレクタのことを指します。


h1 {
	color: black;
}

ユニバーサルセレクタ

要素名の代わりに*を指定すると、すべての要素が適用対象となります。


* {
	color: blue;
}

クラスセレクタ

ピリオドに続けてHTMLのclass属性の値を指定すると、そのclass属性の値を持つ要素が適用対象となります。
class属性の値は空白文字で区切って複数指定できますが、そのうちの1つと一致するだけで適用対象となります。


p.hoge {
	color: white;
}

IDセレクタ

#に続けてHTMLのid属性の値を指定すると、そのid属性の値を持つ要素が適用対象となります。


p#hoge {
	color: red;
}

属性セレクタ

特定の属性が指定されている要素、または特定の属性に特定の値が指定されている要素を適用対象として指定できる。CSS3では次の7種類の属性セレクタが使用できます。

属性セレクタ 適用対象 使用例
[属性名] 「属性名」の属性が指定されている要素 

p[class] {
	color: black;
}
[属性名="属性値"] 「属性名」の属性に「属性値」の値が指定されている要素（値全体が一致） 

a[target="_blank"] {
	color: black;
}
[属性名~="属性値"] 「属性名」の属性に「属性値」の値が指定されている要素（値のどれかと一致でもOK） 

p[class="hoge"] {
	color: black;
}
[属性名|="属性値"] 「属性名」の属性に「属性値」の値が指定されている要素（値全体またはハイフン区切りの前半が一致） 

p[lang|="en"] {
	color: red;
}
[属性名^="属性値の始め"] 「属性名」の属性の値が「属性値の始め」で始まる要素 

a[href=^"#anc-"] {
	color: black;
}
[属性名$="属性値の終わり"] 「属性名」の属性の値が「属性値の終わり」で終わる要素 

a[href$="html"] {
	color: black;
}
[属性名*="属性値の一部"] 「属性名」の属性の値が「属性値の一部」を含む要素 

a[title*="確認"] {
	color: red;
}

擬似要素

タグで囲われている範囲全体をそのまま適用対象にするのではない特殊なセレクタを擬似要素という。

擬似要素 適用対象
::first-line ブロックレベル要素の1行目
::first-letter ブロックレベル要素の1文字目
::before 要素内容の先頭にコンテンツを追加
::after 要素内容の最後にコンテンツを追加

※擬似要素は、結合子も含めたセレクタ全体の最後尾にひとつだけしか付けることができない。先頭のコロンは2つです。

結合子

組み合わせたシンプルセレクタ同士を区切って使用する結合子には、4種類あります。

結合子 適用対象
セレクタA セレクタB Aの要素に含まれているBの要素
セレクタA > セレクタB Aの直接の子要素であるB
セレクタA + セレクタB 共通の親要素を持つ要素の中でAの直後にあらわれるB
セレクタA ~ セレクタB 共通の親要素を持つ要素の中でAのよりも後にあらわれるB

優先順位

CSS適用の優先順位を考える際の基本的な指定元としては「制作者」「ユーザー」「ユーザーエージェント」の3種類が挙げられます。この3つの優先順位は!importantを追加することで変更可能です。

!importantをつけた場合とつけない場合の優先順位をまとめると次のようになります。（上部が優先順位が高く、下部は低い）

  • ユーザーエージェント !important付き
  • ユーザー !important付き
  • 制作者 !important付き
  • 制作者
  • ユーザー
  • ユーザーエージェント

セレクタの詳細度による優先順位の計算

詳細度とは「詳細の度合い」を意味し、要素の種類だけで指定するようなセレクタの詳細度は低く、IDで1つの要素を限定的に指定するセレクタの詳細度は高くなります。

詳細度の計算は3桁の数字で行ないます。セレクタの中に含まれるシンプルセレクタの種類ごとに個数を次のように各桁に入れます。この3桁の数値が大きいものほど詳細度が高いことになり、優先度も高くなります。

  • 1桁目
    タイプセレクタの個数（要素名） ＋ 擬似要素の個数
  • 2桁目
    クラスセレクタの個数 + 属性セレクタの個数 + 擬似クラスの個数
  • 3桁目
    IDセレクタの個数

色

CSSで色を示す場合、16進数・キーワード・rgb()/rgba()・hsl()/hsla()などの方法があります。

16進数

RGBの各値を16進数で指定する書式です。一般的にRGBの各値を2桁ずつ、計6桁で表現します。RGBの各値を示す2桁ずつがそれぞれ同じ数字の繰り返しになっている場合に限り、重複する1桁ずつを省いて3桁で示すことができます。たとえば、#ff0000という値は#f00と書くことができます。

キーワード

CSSの色は英語の色の名前で指定することもできます。150近くのキーワードが定義されています。

基本的な色のキーワード
キーワード 色 16進数での値 キーワード 色 16進数での値
white #fff black #000
gray #808080 silver #c0c0c0
red #f00 maroon #800000
purple #80080 fuchsia #f0f
green #008000 lime #0f0
olive #808000 yellow #ff0
blue #00f aqua #0ff
navy #000080 teal #008080

rgb() rgba()

rgb()という関数形式の書式を使用すると、RGBの値を10進数のまま指定可能です。
RGBの値に加えて不透明度をあらわすAlphaの値も指定できるようにしたのがrgba()という書式です。0.0だと完全に透明、1.0だと完全に不透明になります。

hsl() hsla()

直感的に色を指定したり変更できるようにするために作られたのがhsl()という書式です。hslは、hue（色相）・saturation（彩度）・lightness（明度）の頭文字をとったもので、その名のとおり色相と彩度と明度の順に色を指定します。
彩度と明度に関しては、0%～100%のパーセント値で指定しますが、色相に関しては色相環（カラーサークル）における角度を示す数値で指定します。

さらに、hsl()の値に加えて不透明度をあらわすAlphaの値も指定できるようにしたのがhsla()という書式です。指定の仕方はrgba()同様です。

背景

background-colorプロパティ

ボックスの背景色を設定するプロパティです。背景色をボックスのどの領域に表示させるかは「background-clip」で指定します。このプロパティは、すべての要素に指定でき、初期値はtransparentで透明の状態となっています。

background-imageプロパティ

背景に画像を表示させるプロパティです。値には、「none」または「url()」の書式で画像のURLを指定できます。初期値はnoneです。

CSS3から1つのボックスに複数の背景画像を表示させられるようになっています。URLはカンマで区切って必要なだけ記述します。先に指定した画像ほど上（手前）に表示されます。


body {
	background-image: url(hoge.jpg), url(huga.jpg);
}

補足説明

注意事項

background-color以外のすべての背景関連プロパティでカンマ区切りの複数の値を使えるようになりました。


body {
	background-image: url(hoge.jpg), url(huga.jpg);
	background-repeat: no-repeat, repeat-x;
}

background-imageプロパティで指定した値の数だけレイヤーが生成され、他の背景関連プロパティはそれに対応した数の値を指定できます。レイヤーの数より多く指定した値は無視され、少なく指定された場合は値全体を繰り返す形で適用されます。

グラデーションの指定

背景画像を指定する際、url()の部分にlinear-gradient()またはradial-gradient()という書式を使用することで、画像を使わずにグラデーションを表示させることができます。

background-clipプロパティ

背景ボックスのどの領域に表示させるのかを設定するプロパティです。値の初期値はborder-boxです。値には次のキーワードを指定できます。

値 意味
border-box ボーダー領域を含む、そこから内側の部分だけに背景を表示させる
padding-box パディング領域を含む、そこから内側の部分だけに背景を表示させる
content-box 要素内容を表示させる領域だけに背景を表示させる

補足説明

html要素の背景の表示方法には特別なルールがあります。

  • そのボックスの範囲ではなく、ブラウザなどの表示領域全体に対して適用される（background-clipプロパティを指定しても効果はない）
  • 背景画像の表示サイズや配置位置を指定した場合は、html要素のボックスを基準に計算されますので注意が必要
  • html要素に背景画像がなく、背景色がtransparentの場合、ブラウザはbody要素の背景を代わりに使用することになっている

※仕様書では、ページ全体の背景を指定する際は、html要素ではなくbody要素に指定することを推奨しています。

background-repeatプロパティ

背景画像を縦または横に繰り返して表示させるかを設定するプロパティです。値の初期値はrepeatです。値に指定できるキーワードの数は1つまたは2つです。（2つの値は空白文字で区切って指定します）
値を2つ指定した場合は、1つ目が横方向、2つ目が縦方向に関する繰り返しの指定になります。

値 意味
repeat-x 横方向にのみ画像を隙間なく繰り返し表示させます。
repeat-y 縦方向にのみ画像を隙間なく繰り返し表示させます。
repeat 隙間なく画像を繰り返し表示させます。
space 背景画像の全体が表示される状態で可能なだけ繰り返して表示させます。ピッタリと収まらない場合は、画像と画像の間に隙間ができます。
round 背景画像の全体が表示される状態で可能なだけ繰り返して表示させます。ピッタリと収まらない場合は、画像を拡大縮小して隙間をなくします。
no-repeat 画像を1つだけ表示します。

background-sizeプロパティ

背景画像の表示サイズを設定するプロパティです。初期値はautoです。値に指定できるキーワードの数は1つまたは2つです。（2つの値は空白文字で区切って指定します）
値を2つ指定した場合は、1つ目が幅、2つ目が高さの指定になります。値を1つだけ指定した場合は、幅が指定されたものとみなされ、高さはautoとなります。

値 意味
contain 元の縦横比を保った状態で、背景画像の全体が表示される最大サイズにします。
cover 元の縦横比を保った状態で、背景画像1つで表示領域全体を隙間なく覆う最少サイズにします。
単位つきの数値 数値に単位を付けて指定します。
パーセンテージ 背景の表示領域に対するパーセンテージで指定します。
auto 一方だけががautoの場合、もう一方のサイズに合わせて元の縦横比を保った長さになります。両方がautoの場合は元のサイズで表示します。

background-originプロパティ

背景画像の配置位置を指定する際の基準となる領域を設定するプロパティです。初期値はpadding-boxです。

値 意味
border-box ボーダー領域から内側を配置の基準領域とします。
padding-box パディング領域から内側を配置の基準領域とします。
content-box 要素内容を表示させる領域を配置の基準領域とします。

background-positionプロパティ

背景画像を表示させる領域内での画像の配置位置を設定するプロパティです。画像が繰り返し表示される場合には、まずその位置に画像が配置され、そこから繰り返されることになります。

初期値は0% 0%です。値に指定できるキーワードの数は2つが基本形です。（2つの値は空白文字で区切って指定します）
値を2つ指定した場合は、1つ目が横方向の位置、2つ目が縦方向の位置の指定になります。値を1つだけ指定した場合は、2つ目の値にcenterが指定されたものとして処理されます。

値 意味
単位つきの数値 数値に単位をつけて指定します
パーセンテージ 背景の表示領域に対するパーセンテージで指定します
top 縦方向の0%と同様
bottom 縦方向の100%と同様
center 縦方向の50%、横方向の50%と同様
left 横方向の0%と同様
right 横方向の100%と同様

配置の原点は、background-originプロパティで指定されている配置の基準領域の左上となります。

補足説明

CSS3では3つまたは4つの値が指定できるようになっています。この場合はキーワードによる2つの値が基本となり、それぞれのキーワードのあとに指定する数値またはパーセンテージはキーワードの位置からのオフセットとなります。
たとえば以下のように指定すると、bottomの位置から上へ100px、rightの位置から左へ50pxの位置が指定されたことになります。


body {
	background-position: bottom 100px right 50px;
}

値が3つの場合は、指定していない数値は0となります。正の値を指定すると領域の内側へのオフセットとなりますが、負の値を指定すると外側へのオフセットとなります。

backgroundプロパティ

背景関連のプロパティの値をまとめて指定できるプロパティです。一部の例外を除けば、必要な値を空白文字で区切って順不同で指定できます。

「background-position」と「background-size」は、いずれも単位つきの数値とパーセンテージなので両方がそのまま指定すると区別がつきません。そのため、値が1つだけ指定されている場合、「background-position」の値だと解釈するルールになっています。
「background-size」の値を指定するには、まず「background-position」の値を書き、スラッシュで区切って「background-size」の値を書くと正しく認識されます。

また、「background-clip」と「background-origin」に指定できるキーワードは全く同じであるためこちらも区別がつきません。そのため、1つだけ指定されている場合は「background-clip」と「background-origin」両方に適用され、値が2つ指定されている場合は1つ目の値が「background-origin」、2つ目の値が「background-clip」に適用されることになっています。

補足説明

backgroundプロパティの値もカンマで区切ることで、複数の背景画像のレイヤーを指定できます。その際、background-colorプロパティは一番下のレイヤー（書式上はカンマで区切られた一番最後の値）にしか指定できません。


body {
	background: url(hoge.jpg) no-repeat,
			     url(huga.jpg) no-repeat #fff bottom right / 100% auto;
}

text-shadowプロパティ

テキストに影を表示させるプロパティです。初期値はnoneです。

値 意味
単位つきの数値 数値に単位をつけて指定します。数値は2～3個指定でき、用途は順番により異なります。影の位置とぼかす範囲が指定できます

  • 3番目の数値は、影をぼかす範囲を示します。大きな数値ほどぼかしが強くなり、0だとぼかしがなくなります。負の値は指定できません。
  • 色の値は一番後ろに指定できます。

p {
	text-shadow: 2px 2px 4px rgba(0, 0, 0, .7);
}

Webフォント

CSSでフォントの種類が指定されていても、ユーザの使用環境にそのフォントがインストールされていなければそのフォントで表示させることはできません。しかし、 @font-faceという書式を使用してWeb上にあるフォントを指定することで、そのフォントがインストールされていない環境でもそのフォントが表示できるようになります。

@font-face {

p {
	font-family: mynaviFont, serif;
}

段と段の間隔を設定するプロパティです。値は単位つきの数値で指定します。初期値はnormalです。

段と段の間隔の中央にはボックスのボーダーと同様に線を表示させることができます。その線種・色・太さを個別に設定するのが、次の3つのプロパティです。順不同で指定できます。

  • column-rule-styleプロパティ
  • column-rule-colorプロパティ
  • column-rule-widthプロパティ

これらの値をまとめて設定できるのがcolumn-ruleプロパティです。必要な値を順不同で設定できます。

column-spanプロパティ

段組が設定されているボックスの幅いっぱいに（すべての段をまたいで）表示させるプロパティです。値にはallとnone（初期値）が指定できます。

フレキシブルボックスレイアウト

「display:flex;」を指定された要素は、その内部の子要素を横にも縦にも逆順にも自由に配置することが可能になります。関連するプロパティも多くあり、子要素の配置順序や均等に配置することも可能です。

内部の子要素の並び方は「flex-direction」プロパティで設定します。初期値がrow（横方向）となっているので、display:flex;を指定するだけで子要素は左から右へと順に横並びになります。

アニメーション

回転・拡大縮小・移動

CSS3のtransformプロパティを使うと、要素のボックスを回転させたり、拡大縮小させたり、移動させることができます。それらは三次元空間でも行なえるようになっているため、立体的な3D表現も可能です。ただし、transformプロパティを使用しただけでは、動きを伴わない最終的な表示結果しか見ることができません。変化の過程をアニメーションのように見せるためには、トランジションまたはアニメーションの機能と組み合わせる必要があります。

transformプロパティ

要素のボックスを回転・拡大縮小・移動・変形させるプロパティです。値は関数形式になっており、空白文字で区切って必要なだけ指定できます。ただし、それらの関数は先頭から順に実行されますので、指定順序によって表示結果が変わることもある点に注意してください。初期値はnoneです。

transformプロパティに指定できる値（2Dのみ）
値 意味
none 変形しない
rotate(角度) 時計回りに回転する角度を数値（deg、radなど）で指定
scale(x方向, y方向) x方向、y方向に拡大・縮小する倍率を数値（単位なし）で指定
scaleX(x方向) x方向に拡大・縮小する倍率を指定
scaleY(y方向) y方向に拡大・縮小する倍率を指定
translate(x方向, y方向) x方向、y方向に移動する距離を数値（pxなど）で指定
translateX(x方向) x方向移動する距離を数値で指定
translateY(y方向) y方向移動する距離を数値で指定
skew(x方向, y方向) x軸、y軸に沿って傾斜する角度を数値（deg、radなど）で指定
skewX(x方向) x軸に沿って傾斜する角度を指定
skewY(y方向) y軸に沿って傾斜する角度を指定
matrix(数値･･･) 6つの値からなる行列により二次元の変形を行なう

単位 説明

grad グラード（円周の1/400を1とする単位）

transformには以下のような3D（3次元）用の関数も用意されています。x軸（水平方向）とy軸（垂直方向）に、z軸（奥行き）が加わります。

  • rotate3d（xベクトル, yベクトル, zベクトル, 角度）、rotateX（x軸を軸とする角度）、rotateY（y軸を軸とする角度）、rotateZ（z軸を軸とする角度）
  • translate3d（x方向, y方向, z方向）、translateZ（z方向）
  • scale3d（x方向, y方向, z方向）、scaleZ（z方向）

transform-originプロパティ

transformプロパティで回転・拡大縮小・移動・変形を行なう際の原点を設定するためのプロパティです。初期値は50% 50%で、ボックスの中心が原点となっています。

値 意味
単位つきの数値 ボックスの左上からの距離に単位をつけて指定

CSS3のトランジション

連続した動きでなめらかに変化させるために使用するのがトランジションです。

トランジションは、ある状態から別の状態へという二点間の変化しか表現することができません。それを連続して次々と別の状態へと変化させられるようにしたのがCSS3のアニメーションです。

transition-propertyプロパティ

「どのプロパティ」の値が変化されたときにトランジションを実行させるのかを設定するプロパティです。初期値はallです。

値 意味
プロパティ名 値が変更されたときにトランジションを適用するプロパティ名を指定。カンマで区切って複数指定できます。
all トランジションの適用が可能なすべてのプロパティに適用します
none トランジションを適用しません 

div {
	transition-property: transform, opacity;
}

transition-durationプロパティ

トランジションを実行する際にどれだけの時間をかけて変化させるのかを設定するプロパティです。値にはs（秒）もしくはms（ミリ秒）を指定できます。初期値は0sです。


div {
	/* transformには2秒かけ、opacityには5秒かけます */
	transition-property: transform, opacity;
	transition-duration: 2s, 5s;
}

transition-timing-functionプロパティ

加速したり減速したりして変化のタイミングを調整します。主に次のような値を指定できます。初期値はeaseです。

値 意味
ease ゆっくり始まり、加速し、ゆっくり終わる
ease-in ゆっくり始まり、一定速度で終わる
ease-out 一定速度で始まり、ゆっくり終わる
ease-in-out ゆっくり始まり、ゆっくり終わる
liner 一定速度で変化する

transition-delayプロパティ

トランジションの開始を遅らせるプロパティです。値には、時間を表す数値に単位をつけて指定します。初期値は0sです。

transitionプロパティ

トランジション関連のプロパティの値をまとめて指定できるプロパティです。空白文字で区切って順不同で指定できます。ただし、時間を表す値については1つ目がtransition-durationの値で、2つ目がtransition-delayの値だと認識されます。

CSS3のアニメーション

CSS3のアニメーションはトランジションを連続して実行させるようなものです。
それを実現させるために、どの段階でどの値が変化するのかを記したキーフレームという特別な書式を使用します。

@keyframes

決められた時間の中で、どのタイミングでどの値がどう変化するかをキーフレームと呼ばれる書式で記入します。
■%{プロパティ:値;}の書式で、プロパティと値はいくつでも指定可能です。0%{}と100%{}またはfrom{}とto{}の指定は必須ですが、その間の%は自由に指定できます。


/* 「hoge」は任意の名前。「animation-name」の値に使用する */
@keyframes hoge {
	0% {
		プロパティ: 値;
		プロパティ: 値;
	}
	50% {
		プロパティ: 値;
	}
	100% {
		プロパティ: 値;
	}
}

animation-nameプロパティ

キーフレームを名前で指定して実行させるためのプロパティです。指定できる値はキーフレームの名前またはnone（初期値）です。キーフレームの名前はカンマで区切って複数指定できます。

animation-durationプロパティ

アニメーションの再生時間を設定するプロパティです。値には時間を指定します。初期値は0sです。

animation-timing-functionプロパティ

アニメーションの再生速度の変化パターンを設定するプロパティです。指定できる値は次の通りで初期値はeaseです。

値 意味
ease ゆっくり始まり、加速し、ゆっくり終わる
ease-in ゆっくり始まり、一定速度で終わる
ease-out 一定速度で始まり、ゆっくり終わる
ease-in-out ゆっくり始まり、ゆっくり終わる
liner 一定速度で変化する

animation-delayプロパティ

アニメーション再生の開始を遅らせるプロパティです。値には時間を表す数値に単位をつけて指定します。初期値は0sです。

animation-iteration-countプロパティ

アニメーションを何回繰り返して再生させるのかを設定するプロパティです。初期値は1です。infinitを指定すると、停めるかウィンドウを閉じるまで再生を繰り返します。

animation-directionプロパティ

逆再生させるか、または繰り返し再生する中でどのタイミングで逆再生させるかを設定するプロパティです。初期値はnormalです。

値 意味
normal キーフレーム通りに再生します
reverse 逆再生します
alternate 繰り返しの際、キーフレーム通りの再生と逆再生を順に繰り返します
alternate-reverse 繰り返しの際、逆再生とキーフレーム通りの再生を順に繰り返します

animation-play-stateプロパティ

アニメーションの再生を一時停止させる際に使用するプロパティです。pausedを指定すると一時停止し、runningを指定すると再生が始まります。初期値はrunningです。

animation-fill-modeプロパティ

animation-delayによって再生の開始が遅延されている間の表示、および再生終了後の表示を設定するプロパティです。次の値が指定でき、初期値はnoneです。

属性名 意味
forwards 再生後は@keyframesの100%の表示のままにします
backwards animation-delayの間は@keyframesの0%の表示になります
both animation-delayの間は@keyframesの0%の表示、再生後は100%の表示にします
none @keyframes{}内の指定とは無関係に表示します

animationプロパティ

アニメーション関連のプロパティの値をまとめて指定できるプロパティです。空白文字で区切って順不同で指定できます。時間を表す値については1つ目がanimation-durationの値で、2つ目がanimation-delayの値となります。

その他

直線状のグラデーション

CSSの書式で画像が指定可能なところであれば、url()の代わりにlinear-gradient()またはradial-gradient()という関数を使用してグラデーションを表示させることができます。

linear-gradient()は、上から下、右から左といった直線的なグラデーションを表示させる際に使用します。
radial-gradient()は、まるく放射状のグラデーションを表示する際に使用します。

linear-gradient()関数の基本的な指定方法は次のようになっています。


linear-gradient(方向, 開始色, 終了色)
  • 方向は、「単位をつけた角度」または「to」に続く「left」「right」「top」「bottom」のキーワードで指定します。「0deg」だと下から上、「90deg」だと左から右方向へのグラデーションになります。キーワードは「to top」が下から上、「to right」だと左から右方向へのグラデーションになります。「to top right」のようにして、左下から右上方向へのグラデーションを指定することも可能です。方向は省略が可能で初期値は「to bottom」（上から下方向）です。
  • 開始色と終了色の後ろには、その色になる位置も指定できます。位置は、空白文字で区切って「単位をつけた実数」または「パーセンテージ」で指定できます。

次に示す例はどれも同じ結果になります。


div {
	background: linear-gradient(#ddd, #333);
	background: linear-gradient(to bottom, #ddd, #333);
	background: linear-gradient(180deg, #ddd, #333);
	background: linear-gradient(180deg, #ddd 0%, #333 100%);
}

表示例

放射状のグラデーション

radial-gradient()関数の基本的な指定方法は次のようになっています。


radial-gradient(中心の位置, 中心の色, 外側の色);

中心の位置は、「at」に続けてbackground-positionプロパティと同じ値が指定できます。初期値はcenterです。
中心位置の前に円の種類「circle（正円）」「ellipse（楕円）」、円の大きさも指定できます。また、linear-gradient()と同様に途中の色も追加可能で、色の後ろには位置も指定できます。

次に示す例はどれも同じ結果になります。


div {
	background: radial-gradient(#ddd, #333);
	background: radial-gradient(at center, #ddd, #333);
	background: radial-gradient(at 50% 50%, #ddd, #333);
}
